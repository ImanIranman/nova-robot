# Project Nova — Autonomer mobiler Roboter-Prototyp (V3)

Projekt **Nova** ist eine hochmoderne, autonome Forschungs- und Entwicklungsplattform für mobile Robotik. Das System kombiniert fortschrittliche Algorithmen der künstlichen Intelligenz (Edge AI) mit uneingeschränkter omnidirektionaler Mobilität.

## 🚀 Key Features
* **Zentralgehirn:** NVIDIA Jetson Orin NX (16 GB) für komplexe KI-Modelle und Computer Vision in Echtzeit.
* **Antriebstechnologie:** 4x omnidirektionale Mecanum-Räder auf einer robusten Aluminium-Chassis-Basis.
* **Sensorik:** Hochpräzises LIDAR-System & Intel RealSense Kameras zur dreidimensionalen Raumerkennung und Personendetektion.
* **Steuerung:** ROS-V3 (Robot Operating System) für die präzise, unabhängige Einzelradsteuerung.
* **Energieversorgung:** Parallel geschaltete 12V Li-Ion Akkus für ausgedehnte Laufzeiten.

## 📁 Repository-Struktur

* `/bilder` — Bildmaterial der verschiedenen Entwicklungsstufen (Prototyp, Body, Akku-Integration etc.)
* `index.html` — Portfolio- und Demonstrations-Webseite (Dark Mode)
* `style.css` — Modernes, responsives Styling der Benutzeroberfläche
* `script.js` — Clientseitige Logik der Anwendung
* `nova-dokumentation-v3.pdf` — Ausführliche technische Dokumentation (Schaltpläne, Kinematik & Codebeispiele)

## 🛠️ Lokale Inbetriebnahme der Demo-Webseite

Um die zugehörige Portfolio-Webseite lokal auf deinem Rechner anzuzeigen:

1. Klicke oben rechts auf dieser GitHub-Seite auf **Code** -> **Download ZIP** (oder klone das Repo via Git).
2. Entpacke die ZIP-Datei auf deiner Festplatte.
3. Doppelklicke auf die Datei `index.html`, um die Webseite direkt in deinem Webbrowser zu öffnen.

> **Hinweis zur PDF-Anzeige:** Die interaktive PDF-Dokumentation wird direkt im Browser-Iframe gerendert. Stelle sicher, dass sich die Datei `nova-dokumentation-v3.pdf` im selben Verzeichnis wie die `index.html` befindet.
